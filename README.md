# RustGB

A Game Boy / Game Boy Color Emulator written in Rust

## Quickstart Guide

You can run `rustgb --help` which will output:

```
RustGB - v0.1

USAGE:
  rustgb <filename>

ARGS:
  <filename>          Sets the ROM file to load.
```
